# Security Policy

## Reporting a Vulnerability

DeltaLens is a read-only CLI tool. It does not make network requests, execute arbitrary code, or connect to external services by default.

If you discover a security vulnerability (e.g., path traversal when processing crafted Delta log files), please **do not open a public issue**. Instead, email the maintainers directly or open a draft security advisory on GitHub.

We will respond within 48 hours and coordinate a fix before public disclosure.

## What to expect

- You will receive acknowledgment of your report within 2 business days
- We will provide a timeline for the fix and release
- You will be credited in the release notes (unless you prefer anonymity)
